# Directions and Navigation

| Cantonese     | English            | Notes                        |
| ------------- | ------------------ | ---------------------------- |
| jor           | Left               | Direction                    |
| yow           | Right              | Direction                    |
| zik           | Straight           | Direction                    |
| ore oi jor    | I want left        | Indicating left direction    |
| nay oi yow ma | Do you want right? | Asking about right direction |
