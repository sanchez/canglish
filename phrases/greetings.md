# Greetings and Basic Interactions

| Cantonese            | English           | Notes                                 |
| -------------------- | ----------------- | ------------------------------------- |
| hi                   | Hi                | Simple greeting                       |
| ore ho aaa           | I am good         | Response to "how are you"             |
| nay ho ma            | Are you good?     | Asking how someone is                 |
| mmm ho               | Not good          | Negative response                     |
| ore hai lay pang-yau | I am your friend  | Stating friendship                    |
| nay gew ma           | What's your name? | Asking someone's name                 |
| joy ghin             | Good bye          | Farewell                              |
| mmm goi              | Please            | Polite request                        |
| ore oi               | I want            | Expressing desire (incomplete phrase) |
| nay oi ma            | Do you want?      | Asking if someone wants something     |
| ore chung-ee         | I like            | Expressing preference (incomplete)    |
| nay chung-ee ma      | Do you like?      | Asking if someone likes something     |
| hai                  | Yes               | Affirmative response                  |
